# 2FA

Manage your Two-Factor Authentication (2FA) codes (based on TOTP).

## Features

- Manage 2FA codes
- Auto refresh
- Copy 2FA codes to clipboard
- Dark mode support
- Responsive design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
